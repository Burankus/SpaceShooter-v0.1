# SpaceShooter-v0.1

RMIT Game Studio Project

To run:

- Download the file and import it in Godot v4.4.1

### Gameplay:

- Destroy enemies and grab power-ups drop from them.
- Survive for as long as possible while getting points

Keyboard:
- Movement: Arrow keys
- Shoot: Space

Controller:
- Move: Left stick
- Shoot: A
